# rpncalc-go

[![Build Status](https://travis-ci.org/russmack/rpncalc-go.svg?branch=master)](https://travis-ci.org/russmack/rpncalc-go)

## Usage
```
go build
./rpncalc-go "182 7 27 2 3 4 5 + + + - * /"
```
